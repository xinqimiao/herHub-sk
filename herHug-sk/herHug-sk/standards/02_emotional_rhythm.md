
# 情绪节律评分标准

## 核心概念
情绪节律是用户在时间维度上的情绪变化模式，包括日周期、周周期和恢复力。

## 一、日模式分析

### 高峰时段识别
**定义**：一天中情绪最积极、能量最高的时段

**判断标准**：
- 情绪值 > 0.7（喜悦、兴奋）
- 能量水平 > 0.6
- 持续至少1小时

**输出格式**：
```json
{
"peakHours": [9, 10, 11],
"peakType": "morning",  
"peakIntensity": 0.8
}
```
### 低谷时段识别
**定义**：一天中情绪最消极、能量最低的时段

**判断标准**：
情绪值 < 0.3（悲伤、愤怒、恐惧）
或能量水平 < 0.3
持续至少1小时

**输出格式**：
```json
{
"lowHours": [3, 4],
"lowType": "late-night",
"lowIntensity": 0.7
}
```
**模式分类**

晨型人：高峰在6-12点，低谷在22-4点

夜型人：高峰在18-2点，低谷在6-12点

平稳型：无明显波动（波动度<0.3）

双峰型：上午和晚上两个高峰

波动型：多次起伏（波动度>0.6）

## 二、周模式分析
### 周末效应
**定义**：周末（周六、周日）与工作日（周一至周五）的情绪差异

计算公式：

```text
周末效应 = 周末平均情绪值 - 工作日平均情绪值
```
**解读**：

0.2：明显的周末放松

0 ~ 0.2：轻微周末效应

< 0：工作日反而更开心（可能工作热情高或周末压力大）

### 周一现象

**定义**：周一上午（8-12点）的情绪状态

**判断标准**：

与周二至周四同期相比

情绪值低0.3以上 → 明显周一现象

情绪值低0.1-0.3 → 轻微周一现象

### 周期长度
**定义**：情绪从低谷到高峰的完整周期天数

计算：基于最近7-30天的情绪序列，识别周期性模式

## 三、恢复力指标
**定义**
从负面情绪中恢复到正常水平的速度和能力

恢复速度
计算公式：

```text
恢复速度 = 1 / (负面情绪持续时间（小时）)
```
分级：

高恢复力 (>0.5/小时)：负面情绪<2小时消散

中恢复力 (0.2-0.5/小时)：2-5小时消散

低恢复力 (<0.2/小时)：>5小时或持续到次日

恢复质量
完全恢复：恢复到甚至超过基线水平

部分恢复：恢复到基线但低于正常

累积效应：连续多日负面情绪未完全恢复

## 四、置信度评估
数据充分性
高置信度 (0.7-1.0)：有至少7天数据，每天有5+个采样点

中置信度 (0.4-0.7)：有3-6天数据，每天3-5个采样点

低置信度 (0-0.4)：数据不足3天，或采样稀疏

输出示例
```json
{

  "patternType": "morning",
"peakHours": [9, 10, 11],
"lowHours": [3, 4, 5],

"speed": 0.6,

},

}
```
